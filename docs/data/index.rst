.. module:: data

Data object
-----------

The :class:`Data <NF4HEP.Data>` class acts as a data container. It is used by the :mod:`DNNLikelihood <dnn_likelihood>` object 
and the :mod:`DNNLikEnsemble <dnn_likelihood_ensemble>` object to manage training/validation/test data
through the :class:`Data.data_dictionary <NF4HEP.Data.data_dictionary>` dictionary.
See the documentation of the :mod:`DNNLikelihood <dnn_likelihood>` object and 
the :mod:`DNNLikEnsemble <dnn_likelihood_ensemble>` object for more details on how the
:mod:`Data <data>` object is used when building the NF4HEP.
