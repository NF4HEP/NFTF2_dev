.. module:: NF

NF object
---------

The :class:`NF <NF4HEP.NF>` is the key object in the frameword and represents the DNNLikelihood in the form
of a DNN. The object includes all methods to build the DNN, optimize it, and use it for inference. All data in the object
is managed by the :mod:`Data <data>` module.
